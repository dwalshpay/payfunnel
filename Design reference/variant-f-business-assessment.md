# Variant F: The Business Rewards Assessment

> **Strategic Focus**: Qualified leads through professional positioning and natural qualification

---

## Concept Overview

### The Big Idea
Reframe the signup funnel as a **professional diagnostic tool**. Users complete a "Business Rewards Assessment" and receive a scored report. This attracts serious business owners, naturally qualifies leads through diagnostic questions, and creates commitment through invested effort.

### Why This Works
The "assessment" framing changes the user's mental model:

1. **Authority Positioning**: "Assessment" suggests expertise and diagnostic capability
2. **Commitment Escalation**: Each completed section increases investment and likelihood of finishing
3. **Curiosity Gap**: The teased report creates desire to see results
4. **Social Comparison**: Benchmark against peers triggers competitive drive
5. **Reciprocity**: Providing a valuable report creates obligation to consider the offer

### Target User
- Serious business owners evaluating options (not casual browsers)
- Decision-makers who appreciate data-driven insights
- Users who want to understand their specific situation
- High-value prospects with genuine business payment needs

### Qualification Signals
The assessment captures signals that indicate lead quality:

| Signal | Weight | High Value Indicator |
|--------|--------|---------------------|
| Monthly expenses | 30% | $50K+ per month |
| Industry | 20% | Construction, trades, property |
| Pain points | 25% | Suppliers won't accept cards, ATO issues |
| Team size | 15% | 20+ employees |
| Rewards goals | 10% | Travel focus (higher LTV) |

### Score Calculation

```typescript
interface AssessmentScore {
  total: number;          // 0-100
  tier: 'gold' | 'silver' | 'bronze';
  percentile: number;     // Top X%
}

const calculateScore = (answers: AssessmentAnswers): AssessmentScore => {
  let score = 0;

  // Section 1: Business Profile (max ~35 points)
  score += industryScores[answers.industry];      // 10-20
  score += teamSizeScores[answers.teamSize];      // 5-20
  score += businessAgeScores[answers.businessAge];// 5-15

  // Section 2: Payment Habits (max ~60 points)
  score += expenseScores[answers.monthlyExpenses];// 5-35
  score += answers.paymentMethods.reduce(         // 1-13
    (sum, method) => sum + paymentMethodScores[method], 0
  );
  score += painPointScores[answers.painPoint];    // 5-25

  // Section 3: Rewards Goals (max ~18 points)
  score += answers.rewardsGoals.reduce(           // 4-28
    (sum, goal) => sum + goalScores[goal], 0
  );

  // Normalize to 100
  const normalizedScore = Math.min(Math.round((score / 116) * 100), 100);

  // Determine tier
  let tier: 'gold' | 'silver' | 'bronze';
  if (normalizedScore >= 90) tier = 'gold';
  else if (normalizedScore >= 70) tier = 'silver';
  else tier = 'bronze';

  // Calculate percentile (higher score = lower percentile = better)
  const percentile = Math.max(5, 100 - normalizedScore);

  return { total: normalizedScore, tier, percentile };
};
```

const generateRecommendations = (answers: AssessmentAnswers): string[] => {
  const recommendations: string[] = [];

  // Based on pain point
  if (answers.painPoint === 'no_card_acceptance') {
    recommendations.push(
      'Pay suppliers via pay.com.au with your credit card, even when they only accept bank transfers'
    );
  }

  if (answers.painPoint === 'ato_reduced') {
    recommendations.push(
      'Pay ATO bills through pay.com.au to earn full points (instead of reduced category points)'
    );
  }

  // Based on goals
  if (answers.rewardsGoals.includes('flights')) {
    const preferredAirline = answers.airlineProgram || 'Qantas';
    recommendations.push(
      `Transfer PayRewards to ${preferredAirline} for maximum flight redemption value`
    );
  }

  if (answers.rewardsGoals.includes('cash_back')) {
    recommendations.push(
      'Use PayRewards to offset business invoices directly - unique to pay.com.au'
    );
  }

  // Based on payment methods
  if (answers.paymentMethods.includes('bank_transfer')) {
    recommendations.push(
      'Earn PayRewards on bank transfers too - get rewarded on payments you\'re already making'
    );
  }

  return recommendations.slice(0, 4); // Max 4 recommendations
};

## Lead Routing Logic

```typescript
interface LeadRoute {
  tier: 'gold' | 'silver' | 'bronze';
  action: 'sales_call' | 'premium_onboard' | 'standard_onboard' | 'self_serve';
  priority: 'high' | 'medium' | 'low';
}

const routeLead = (score: number, answers: AssessmentAnswers): LeadRoute => {
  // Gold tier with high-intent pain point
  if (score >= 90 && ['no_card_acceptance', 'ato_reduced'].includes(answers.painPoint)) {
    return { tier: 'gold', action: 'sales_call', priority: 'high' };
  }

  // Gold tier without specific pain point
  if (score >= 90) {
    return { tier: 'gold', action: 'premium_onboard', priority: 'high' };
  }

  // Silver tier
  if (score >= 70) {
    return { tier: 'silver', action: 'standard_onboard', priority: 'medium' };
  }

  // Bronze tier
  return { tier: 'bronze', action: 'self_serve', priority: 'low' };
};
```
